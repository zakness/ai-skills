# ai-skills

Reusable Claude Code skills shared across multiple projects.

## Skills

### From [mattpocock/skills](https://github.com/mattpocock/skills)

- **grill-me** — Interview the user relentlessly about a plan or design until reaching shared understanding
- **improve-codebase-architecture** — Surface architectural friction and propose module-deepening refactors as GitHub issues
- **prd-to-issues** — Break a PRD into independently-grabbable GitHub issues using tracer-bullet vertical slices
- **prd-to-plan** — Turn a PRD into a multi-phase implementation plan saved as a local Markdown file
- **write-a-prd** — Create a PRD through user interview and codebase exploration, submitted as a GitHub issue
- **request-refactor-plan** — Create a detailed refactor plan with tiny commits, filed as a GitHub issue

### From [antiquesonline](https://github.com/zakness/antiquesonline)

- **commit** — Create focused, atomic commits following the Conventional Commits specification
- **create-pr** — Open a ready-for-review GitHub pull request with a generated title and description
- **code-review** — Review a pull request for correctness, clarity, and adherence to project conventions
